# How Crypto Works

**A comprehensive guide to understanding cryptocurrency technology**

Welcome to the bilingual edition of "How Crypto Works" - a deep dive into the mechanics, markets, and mechanisms that power the cryptocurrency ecosystem.

## About This Book

This GitBook provides both English and Korean versions of each chapter, allowing readers to:
- Switch between languages using tabs
- Compare technical terminology across languages
- Learn cryptocurrency concepts in their preferred language

## Coverage

From Bitcoin's foundational proof-of-work to cutting-edge innovations in DeFi, MEV, and quantum resistance, this book explores:

- **Core Protocols**: Bitcoin, Ethereum, Solana, and L1 blockchains
- **Infrastructure**: Custody, market structure, and trading
- **Advanced Topics**: DeFi, MEV, stablecoins, RWAs
- **Emerging Tech**: NFTs, governance, DePIN, prediction markets
- **Future Challenges**: Quantum resistance

## How to Use This Book

Each chapter contains **English** and **한국어** tabs - simply click to switch between languages. Technical terms are preserved in English with Korean translations provided in parentheses on first use.

## Navigation

Use the sidebar to navigate between chapters. Start with the Preface for an overview, then proceed through the chapters in order or jump to topics of interest.

**Note**: This is a technical reference. Some familiarity with cryptocurrency basics is recommended but not required.
