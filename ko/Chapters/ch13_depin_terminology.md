# 제13장: DePIN - 용어집

## 핵심 개념

| 영어 | 한국어 | 비고 |
|------|--------|------|
| DePIN (Decentralized Physical Infrastructure Networks) | 탈중앙화 물리적 인프라 네트워크 | 물리적 인프라를 구축하고 운영하기 위한 암호경제적 프로토콜 |
| Burn-and-mint equilibrium (BME) | 번-앤-민트 균형 | 네트워크 사용 시 토큰을 소각하고 제공자에게 새 토큰을 발행하는 메커니즘 |
| Proof-of-Coverage | 커버리지 증명 | 무선 네트워크에서 실제 커버리지 제공을 검증하는 메커니즘 |
| Proof-of-Spacetime | 시공간 증명 | 데이터가 시간이 지나도 지속적으로 저장됨을 증명 |
| Proof-of-Replication | 복제 증명 | 데이터의 고유한 사본이 저장소에 존재함을 증명 |
| Proof-of-Render | 렌더 증명 | GPU 렌더링 작업이 올바르게 완료되었음을 검증 |
| Proof-of-Location | 위치 증명 | 장치의 물리적 위치를 검증하는 하드웨어 기반 메커니즘 |

## 경제 메커니즘

| 영어 | 한국어 | 비고 |
|------|--------|------|
| Cold-Start Problem | 콜드 스타트 문제 | 초기 네트워크에 고객이 없는 상황 |
| Token Emissions | 토큰 발행 | 초기 네트워크 부트스트래핑을 위한 토큰 배포 |
| Emission Schedule | 발행 일정 | 토큰이 발행되는 시간표 |
| Halving Schedule | 반감기 일정 | 보상이 주기적으로 절반으로 감소하는 일정 |
| Geographic Multipliers | 지리적 승수 | 서비스 부족 지역에 더 높은 보상을 제공하는 메커니즘 |
| Buy-and-burn | 바이백-앤-번 | 수익으로 토큰을 구매해 소각하는 메커니즘 |
| Storage Deals | 저장 거래 | Filecoin에서 클라이언트와 제공자 간의 저장 계약 |
| Endowment Model | 기부금 모델 | Arweave의 영구 저장을 위한 선불 자금 모델 |
| Centralization Trap | 중앙집중화 함정 | 수익성 높은 지역에만 인프라가 배치되는 현상 |
| Death Spiral | 데스 스파이럴 | 사용량 감소 -> 가격 하락 -> 운영자 이탈의 악순환 |

## 역할

| 영어 | 한국어 | 비고 |
|------|--------|------|
| Service Providers | 서비스 제공자 | 물리적 하드웨어를 배치하고 운영하는 참여자 |
| Validators | 검증자 | 서비스 제공 증거를 확인하는 참여자 |
| Challenge Generators | 챌린지 생성기 | 무작위 검증 작업을 생성하는 시스템 |
| Witness Networks | 증인 네트워크 | 독립적으로 응답을 확인하는 네트워크 |
| Supply Side | 공급 측 | 인프라를 제공하는 운영자들 |
| Demand Side | 수요 측 | 네트워크 서비스를 소비하는 사용자들 |
